# Customer Churn Prediction System

This project predicts whether a telecom customer is likely to churn based on account and service information.

## Features
- Machine Learning churn prediction
- FastAPI backend
- Streamlit frontend
- Real-time predictions
- Probability score output

## Dataset
IBM Telco Customer Churn Dataset

## Technologies Used
- Python
- Scikit-learn
- FastAPI
- Streamlit
- Pandas
- Joblib
